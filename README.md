## hud-enhanced

    <(*)__ <(*)__ <(*)__
     (___/  (___/  (___/
    ~-~-~~-~~-~~-~-~-~~-~

An EXILED plugin that adds a number of custom UI elements to all classes (including spectators).

- Name
- Rank
- Class
- Kill Count
- Spectator Count
- Noise Level
- NoClip State
- Elapsed Round Time
- Wave Countdown
- Next Team Wave
- NTF Tickets
- CI Tickets

## Commands

- **`.hud`**: Toggles the custom UI elements.
